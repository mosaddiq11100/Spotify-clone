# 🎵 Spotify Clone (HTML & CSS)

A simple and responsive **Spotify-inspired music player UI** built using pure **HTML and CSS**.
This project focuses on layout design, styling, and recreating the look and feel of Spotify’s interface.

---

## 📌 Features

* 🎧 Spotify-like UI design
* 📱 Responsive layout (works on different screen sizes)
* 🎨 Clean and modern design using CSS
* 📂 Organized file structure
* 🖱️ Sidebar navigation
* ▶️ Music player footer layout

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* (No JavaScript used)

---

## 📁 Folder Structure

```
spotify-clone/
│── index.html
│── style.css
│── /assets
│    ├── images/
│    ├── icons/
```

## ⚙️ Installation & Usage

1. Clone the repository

```bash
git clone https://github.com/your-username/spotify-clone.git
```

2. Navigate to the project folder

```bash
cd spotify-clone
```

3. Open `index.html` in your browser
